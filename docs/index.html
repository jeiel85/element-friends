화합물 합성

7가지 화합물 레시피

두 원소를 드래그해서 합치면 실제 화학 반응이 일어납니다.

💧
H₂O
물 (Water)
H + O
생명에게 가장 소중하고 깨끗한 물이에요! 수소와 산소가 합체해서 투명한 방울이 되었답니다.
🫧
CO₂
이산화탄소 (Carbon Dioxide)
C + O
식물들의 맛있는 밥이자, 우리가 숨을 내쉴 때 나가는 뽀글뽀글 거품 가스예요.
🧂
NaCl
소금 (Salt)
Na + Cl
음식을 아주 맛있게 만들어 주는 하얗고 보배로운 소금이에요! 바다에 아주 많답니다.
🧪
HCl
염산 (Hydrochloric Acid)
H + Cl
우리 위장 속에서 피자나 밥을 골고루 소화시키는 용감한 액체예요!
💨
NH₃
암모니아 (Ammonia)
H + N
약간 찌릿한 냄새를 풍기지만, 농부 아저씨의 밭을 기름지게 자라게 돕는 영양소랍니다.
🧱
Fe₂O₃
녹 (Rust)
Fe + O
단단한 철이 산소 공기랑 친구가 되면서 오렌지색으로 변신하는 귀여운 결합체예요.
🐮
CH₄
메탄가스 (Methane)
C + H
젖소들이 뿡~ 하고 뀌는 방귀나 주방 가스렌지에 들어 있어 깨끗한 열에너지를 만들어 줘요.

주요 기능

즐겁게 배우는 화학

드래그 합성부터 일일 퀴즈까지, 모든 기능이 어린이를 위해 설계되었습니다.

🧪
실험실 (Lab Mode)
원소 버블을 드래그해서 합성하면 화합물이 탄생합니다. 콤보를 이어나가 더 많은 점수를 획득하세요!
📅
일일 퀴즈 (Daily Quiz)
날짜 기반으로 매일 새로운 화학 퀴즈가 출제됩니다. 스트릭을 이어나가 꾸준한 과학자 배지를 받으세요!
🏆
배지 시스템 (Badges)
8가지 성취 배지를 모아 아기 연금술사에서 웅장한 원소 박사까지 성장하세요!
📚
도감 (Compendium)
발견한 화합물이 자동으로 도감에 기록됩니다. 7종 모두 발견해 컬렉션을 완성하세요!
🔊
사운드 이펙트
합성 성공, 콤보, 퀴즈 정답 시 귀여운 효과음이 재생됩니다. AudioTrack 기반 절차적 사운드!
🌈
파스텔 디자인
Material Design 3 기반의 어린이 친화적 파스텔 테마. 8가지 원소마다 고유한 색상!
